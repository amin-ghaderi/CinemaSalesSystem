# 🎬 CinemaSalesSystem — Release Notes

## Version 1.0.0
**Release Date:** April 11, 2026

### Overview
Initial production release of the CinemaSalesSystem, built using Clean Architecture and Lightweight Domain-Driven Design (DDD).

### Features
- Cinema ticket sales management
- Console-based user interface
- PostgreSQL and EF Core support
- Docker containerization
- GitHub Actions CI/CD integration
- Automated testing and validation

### Technical Stack
- .NET 8
- C#
- Entity Framework Core
- PostgreSQL
- Docker
- GitHub Actions

### Architecture
- Clean Architecture
- Lightweight Domain-Driven Design (DDD)
