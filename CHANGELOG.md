## [1.0.0] - 2026-04-11

### Added
- Initial production release of CinemaSalesSystem.
- Console-based user interface for cinema sales management.
- Implementation based on Clean Architecture.
- Lightweight Domain-Driven Design (DDD) principles.
- PostgreSQL integration using Entity Framework Core.
- In-memory database support for testing.
- Docker containerization and Docker Compose orchestration.
- GitHub Actions CI/CD pipelines for automated builds and releases.
- Comprehensive unit and infrastructure tests.
- Automated database migrations and seeding.
- Structured logging using Serilog.
- Production-ready deployment configuration.
- Complete technical, architectural, and user documentation.

### Infrastructure
- Multi-stage Dockerfile for optimized container builds.
- Release packaging and checksum generation.
- GitHub Release automation with version tagging.

### Documentation
- User Guide, Technical Documentation, and Architecture Documentation.
- Deployment Guide and Final Delivery Report.
- Release Notes for version v1.0.0.
- Deployment Validation Report.

### Security
- Secure configuration management using environment variables.
